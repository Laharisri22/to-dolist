# ✅ To-Do List App

A simple and interactive **To-Do List Application** that helps users organize and manage their daily tasks efficiently.
Users can add tasks, mark them as completed, and delete tasks once they are finished.

---

## 🚀 Features

* ➕ Add new tasks
* ✔️ Mark tasks as completed
* 🗑️ Delete tasks from the list
* 📋 Display all tasks in a clean list format
* 💾 Store tasks using browser Local Storage *(optional if implemented)*
* 📱 Simple and responsive user interface

---

## 🛠️ Technologies Used

* **HTML** – For creating the structure of the application
* **CSS** – For styling and layout design
* **JavaScript** – For adding functionality and task management

---

## ⚙️ How It Works

1. Enter a task in the input field.
2. Click the **Add button** to add the task to the list.
3. Click on a task to mark it as **completed**.
4. Click the **delete icon/button** to remove the task.
